ℹ Sobre

Sobre a Plataforma

Informações técnicas e estruturais da plataforma Oficina do Amanhã.

🛠️

OficinaAdmin

Ambiente operacional para professores e administradores. Gestão de turmas, chamadas, horas e distribuição de conteúdo pedagógico.

🧩

OficinaCode

Ambiente do aluno. Ferramenta de programação em blocos com compilação para microcontroladores e acesso a trilhas de aprendizagem.

📚

Central de Conteúdo

Trilhas pedagógicas por projeto e tipo de escola. Material completo para que professores sem formação técnica conduzam aulas de robótica.

🔐

Controle de Acesso

Sistema RBAC com perfis distintos — Administrador, Professor e Aluno. Cada perfil acessa apenas os dados e funções do seu escopo.

Stack técnica

OficinaAdmin Java 21 + JavaFX
OficinaCode Tauri 2 + React + Rust
Banco de dados PostgreSQL via Supabase
Controle de acesso RBAC — Role Based Access Control
Plataforma Windows 10+ (64-bit)
Sprint atual Sprint 1 — v1.0.0
