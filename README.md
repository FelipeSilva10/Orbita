# 🌐 Oficina Portal

Portal web interno da plataforma **Oficina do Amanhã**.

Página de acesso restrito para download das aplicações, acompanhamento de atualizações e suporte técnico.

---

## O que é

O Oficina Portal é o ponto central de distribuição da plataforma. Por ele, membros autorizados fazem login, baixam as versões mais recentes do OficinaAdmin e do OficinaCode, e acompanham o histórico de atualizações.

Não requer framework, build tool ou servidor especial — é um único arquivo HTML.
